Bolbosh: A Multi-Speaker Text-to-Speech System for Kashmiri

Tajamul Ashraf1,3,*, Burhaan Rasheed Zargar3,†, Saeed Abdul Muizz3,†, Ifrah Mushtaq2, Nazima Mehdi2, Iqra Altaf Gillani3, Aadil Amin Kak2, Janibul Bashir3

Abstract

Bolbosh is a multi-speaker text-to-speech (TTS) system for the Kashmiri language (Persio-Arabic script). Adapting the Matcha-TTS framework—a non-autoregressive, conditional flow matching (CFM) based model—Bolbosh synthesizes natural-sounding Kashmiri speech from text. To the best of our knowledge, this is among the first neural TTS systems for Kashmiri, a low-resource language spoken in the Kashmir region.

Our system is trained on 424 speakers by combining the Rasa dataset and IndicVoices corpus. We employ transfer learning from a pre-trained English multi-speaker model (VCTK) and utilize character-level text processing with a custom Kashmiri text normalizer—eliminating the need for a phonemizer. Using an ODE-based inference procedure, Bolbosh enables fast, high-quality synthesis in as few as 10 steps.

Model Architecture Overview

Bolbosh replaces standard phonetic processing with robust character-level representations suitable for Kashmiri orthography, utilizing the Conditional Flow Matching (CFM) objective proposed in Matcha-TTS.

Key Components

  • Text Encoder: 6-layer Transformer with Rotary Positional Embeddings (RoPE), 2 attention heads, 192 hidden channels.
  • Text Processing: Custom KashmiriNormalizer handling Unicode canonicalization, diacritic preservation, and Plat Ye corrections.
  • Vocabulary: 272 character tokens bypassing any phonetic tool (phonemizer-free).
  • Duration Predictor: 2-layer CNN estimating character timing.

Synthesis Engine

  • Flow Matching Decoder: 1-D U-Net using SnakeBeta activation, predicting the optimal vector field.
  • Speaker Conditioning: 64-dimensional embeddings (learned jointly over 424 speakers).
  • Vocoder: HiFi-GAN universal v1 reconstructing waveforms from 80-bin mel-spectrograms.
